Files AeliusBRUBT.pkl and AeliusRUBT.pkl contain statistical models of Portuguese for POS-tagging using NLTK and/or Aelius, as it is the case of the other models described belolw. These two models were trained on a prunned, partly retagged and partly retokenized version of the Tycho Brahe Parsed Corpus of Historical Portuguese (TBCHP) [http://www.tycho.iel.unicamp.br/~tycho/corpus/en/index.html].

Reference:
GALVES, C.; FARIA, P. Tycho Brahe Parsed Corpus of Historical Portuguese. [S.l.: s.n.], 2010. On-line: <http://www.tycho.iel.unicamp.br/~tycho/corpus/en/index.html>



Files AeliusHunPosMM and AeliusStanfordMM are statistical language models trained with the HunPos open source POS-tagger [http://code.google.com/p/hunpos/] and the open source StanfordTagger [http://nlp.stanford.edu/software/tagger.shtml], respectively, on the version of the MAC-MORPHO Corpus of Brazilian Portuguese prepared for tagger training [http://www.nilc.icmc.usp.br/lacioweb/macmorpho.php] and adapted for distribution with NLTK. This corpus is part of the Lácio-Web Project [http://www.nilc.icmc.usp.br/lacioweb/index.htm]. The HunPos tagger and the StanfordTagger are described in the following papers, respectively:



HALÁCSY, P.; KORNAI, A. ; ORAVECZ, C. HunPos: an open source trigram tagger. In: ANNUAL MEETING OF THE ASSOCIATION FOR COMPUTATIONAL LINGUISTICS, 45., 2007, Prague. Proceedings... Stroudsburg: Association for Computational Linguistics, 2007. p. 209-212. On-line: <http://acl.ldc.upenn.edu/P/P07/P07-2053.pdf>

TOUTANOVA, K. et al. Feature-Rich Part-of-Speech Tagging with a Cyclic Dependency Network. In: HLT-NAACL, 2003, Edmonton. Proceedings... [S.l.: s.n.], 2003. p. 252-259. On-line: <http://nlp.stanford.edu/~manning/papers/tagging.pdf>



File AeliusHunPos is a statistical language model trained with the HunPos open source POS-tagger on the above-mentioned version of the TBCHP.



Folders AeliusMaxEnt and AeliusMaxEntMM are statistical models of Portuguese trained on the TBCHP and the MAC-Morpho Corpus, respectively, with the MXPOST software [ftp://ftp.cis.upenn.edu/pub/adwait/jmx/jmx.tar.gz], described in the following paper:



RATNAPARKHI, A. A Maximum Entropy Model for Part-Of-Speech Tagging. In: EMPIRICAL METHODS IN NATURAL LANGUAGE PROCESSING, 1996, Philadelphia. Proceedings... Pennsylvannia: University of Pennsylvannia, 1996. p. 133-142. On-line: <http://acl.ldc.upenn.edu/W/W96/W96-0213.pdf>



Files AeliusMaxentOpenNLP.bin and AeliusPerceptronOpenNLP.bin are probability models trained with Apache OpenNLP tagger [http://opennlp.apache.org/] on the above-mentioned version of the MAC-Morpho Corpus.



File freq_tycho_a.pkl is a NLTK word frequency distribution of a sample of this corpus containing about 200000 alphabetical tokens.

File example.txt contains an excerpt with the first two paragraphs from the novel Luzia-Homem, published in 1903 by the Brazilian writer Domingos Olímpio (born in Sobral, Ceará, in 1850, and died in Rio de Janeiro in 1906), extracted from Wikisource [http://pt.wikisource.org/wiki/Luzia-Homem/I]. File exemplo.nltk.gold.txt is a manually revised version of the POS-tagged output from the AeliusBRUBT tagger. Both untagged and tagged versions of this excerpt were revised by Leonel F. de Alencar and constitute part of the CORPTEXLIT Project [http://complin.blogspot.com/]. File luzia_inicio.txt (like derived annotated files) is a shorter version of example.txt.
